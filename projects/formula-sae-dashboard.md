Real-time telemetry dashboard for USC Formula SAE Racing Team. Built to monitor live sensor data from the race car during testing and competition runs.

## Overview

The dashboard streams live data from the Formula SAE vehicle — covering throttle position, brake pressure, wheel speeds, suspension travel, and more — and visualizes it in a clean, driver-team-facing interface.

## Stack

- **Next.js** for the web app with server-side data handling
- Secure login screen for team access
- Real-time sensor data display with channel graphs and gauges
